
events.csv: 
the original raw data

processed events.csv: 
the cleaned data for machine learning.

data process.ipynb: 
How the data is processed from events.csv to processed events.csv.

machine learning.ipynb: 
the main machine learning module. Some different results stores as the same parameter.

grid research.ipynb: 
the grid research module, including the plotting module. Parameter changes to plot different figures, so there are only 6 cells.

grid research parameters.txt: the parameter used in grid research.

app.py: 
The android application. Run it to run the application.

functions.py: 
contains some functions used in machine learning.ipynb.

recommend.py 
contains some functions used in app.py.
